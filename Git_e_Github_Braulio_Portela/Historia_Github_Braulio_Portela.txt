a. Quem inventou o sistema de controle de versão Git e por quê?
O Git foi criado por Linus Torvalds em 2005, que se tornou o sistema padrão do mercado. Os modelos de versionamento antigo ficaram muito defasados devido a popularidade da produção de códigos.

b. Quem atualmente é o dono do Github e por quê?
Em 2018, foi comprada por US$7,5Bi pela Microsoft, uma das principais parceiras da plataforma desde 2012. Foi comprada devido ao grande potencial que a plataforma veio aprensentando desde 2010.

c. Existe outra maneira, além do terminal, de trabalhar com o Github?
Sim. É possível utilizar o GitHub Desktop e o GitHub dentro do VSCode.
